## 🔧 Инструкция по загрузке файлов:

### В Excel:
1. Файл → Открыть → выберите нужный CSV файл
2. Если проблемы с кодировкой: Данные → Получить данные → Из файла → Из текста/CSV
3. Настройте: кодировка UTF-8, разделитель ";"
4. Загрузить → Закрыть

### В Power BI:
1. Получить данные → Текст/CSV  
2. Выберите файл → Загрузить
3. Для связанных таблиц: Модель → Создать связи
4. Перетащите поля между таблицами для автосвязывания

## ⚠️ Частые проблемы:

### Excel показывает иероглифы:
1. Откройте файл в блокноте
2. Сохранить как → кодировка UTF-8  
3. Теперь откройте в Excel

### Python не читает CSV:
```python
# Попробуйте разные кодировки
df = pd.read_csv('file.csv', encoding='utf-8-sig')
# или
df = pd.read_csv('file.csv', encoding='cp1251')
```

### Power BI не видит связи:
1. Проверьте одинаковые названия столбцов в таблицах
2. Убедитесь что типы данных совпадают  
3. Используйте Модель → Управление связями
