## 🐍 Задание 3.4: Продвинутая визуализация в Python
**Время выполнения: 35 минут**

### Ситуация
Анализируете данные фитнес-приложения "ЗОЖ-трекер" — активность пользователей по дням недели и времени.

### Подготовка данных (10 минут)

```python
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np

# Настройка русского языка
plt.rcParams['font.family'] = 'DejaVu Sans'

# Данные активности пользователей
np.random.seed(42)

hours = list(range(6, 24))  # с 6 утра до 24 часов
days = ['Понедельник', 'Вторник', 'Среда', 'Четверг', 'Пятница', 'Суббота', 'Воскресенье']

# Создаём реалистичные данные с паттернами
data = []
for day_idx, day in enumerate(days):
    for hour in hours:
        # Базовый уровень активности
        base_activity = 50
        
        # Пики в 7-9 утра и 18-20 вечера
        if hour in [7, 8, 9]:
            morning_boost = 80
        else:
            morning_boost = 0
            
        if hour in [18, 19, 20]:
            evening_boost = 100
        else:
            evening_boost = 0
        
        # Выходные отличаются от будней
        if day_idx >= 5:  # суббота-воскресенье
            weekend_modifier = 0.7 if hour < 10 else 1.2
        else:
            weekend_modifier = 1.0
        
        activity = int((base_activity + morning_boost + evening_boost) * weekend_modifier * np.random.uniform(0.8, 1.2))
        
        data.append({
            'День': day,
            'Час': hour,
            'Активность': activity,
            'День_недели': day_idx
        })

df = pd.DataFrame(data)
print("📊 Данные подготовлены:")
print(df.head())
```
