## 🌐 Задание 4.3: Анализ времени отклика сайта
**Время выполнения: 40 минут**

### Ситуация
IT-отдел интернет-магазина "Быстрые покупки" мониторит производительность сайта. Нужно проанализировать времена отклика и найти проблемы.

### Данные
Время отклика в миллисекундах за месяц работы. Файл `response-times.csv`.

### Шаг 1: Исследование данных в Python (20 минут)

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from scipy import stats

# Загрузка данных
df = pd.read_csv('response-times.csv', encoding='utf-8', sep=';')

# Базовый анализ
print("📊 Основная статистика времени отклика:")
print(df['response_time_ms'].describe())

# Проверка нормальности
response_times = df['response_time_ms']
mean_rt = response_times.mean()
std_rt = response_times.std()
skewness = stats.skew(response_times)
kurtosis = stats.kurtosis(response_times)

print(f"\n🔍 Тесты на нормальность:")
print(f"Среднее: {mean_rt:.2f} мс")
print(f"Медиана: {response_times.median():.2f} мс")
print(f"Стандартное отклонение: {std_rt:.2f} мс")
print(f"Асимметрия: {skewness:.3f} {'✅' if abs(skewness) < 0.5 else '❌'}")
print(f"Эксцесс: {kurtosis:.3f}")

# Правило 68-95-99.7
within_1sigma = ((response_times >= mean_rt - std_rt) & 
                 (response_times <= mean_rt + std_rt)).sum() / len(response_times) * 100
within_2sigma = ((response_times >= mean_rt - 2*std_rt) & 
                 (response_times <= mean_rt + 2*std_rt)).sum() / len(response_times) * 100
within_3sigma = ((response_times >= mean_rt - 3*std_rt) & 
                 (response_times <= mean_rt + 3*std_rt)).sum() / len(response_times) * 100

print(f"\n📏 Правило трёх сигм:")
print(f"В пределах ±1σ: {within_1sigma:.1f}% (норма: 68.2%)")
print(f"В пределах ±2σ: {within_2sigma:.1f}% (норма: 95.4%)")
print(f"В пределах ±3σ: {within_3sigma:.1f}% (норма: 99.7%)")
```

### Шаг 2: Поиск аномалий (10 минут)

```python
# Находим аномальные времена отклика
z_scores = np.abs((response_times - mean_rt) / std_rt)

# Аномалии (|Z| > 3)
severe_outliers = response_times[z_scores > 3]
moderate_outliers = response_times[(z_scores > 2) & (z_scores <= 3)]

print(f"\n⚠️ Найдено аномалий:")
print(f"Серьёзные (>3σ): {len(severe_outliers)} ({len(severe_outliers)/len(response_times)*100:.2f}%)")
print(f"Умеренные (2-3σ): {len(moderate_outliers)} ({len(moderate_outliers)/len(response_times)*100:.2f}%)")

if len(severe_outliers) > 0:
    print(f"Максимальное время отклика: {severe_outliers.max():.0f} мс")
    print(f"Минимальное аномальное время: {severe_outliers.min():.0f} мс")

# SLA анализ (Service Level Agreement)
sla_2000ms = (response_times <= 2000).sum() / len(response_times) * 100
sla_1000ms = (response_times <= 1000).sum() / len(response_times) * 100
sla_500ms = (response_times <= 500).sum() / len(response_times) * 100

print(f"\n📊 SLA анализ:")
print(f"Запросы ≤ 500мс:  {sla_500ms:.1f}%")
print(f"Запросы ≤ 1000мс: {sla_1000ms:.1f}%")
print(f"Запросы ≤ 2000мс: {sla_2000ms:.1f}%")
```

### Шаг 3: Визуализация и рекомендации (10 минут)

```python
# Создаём комплексную визуализацию
fig, axes = plt.subplots(2, 2, figsize=(15, 12))

# 1. Гистограмма с нормальной кривой
axes[0, 0].hist(response_times, bins=50, density=True, alpha=0.7, color='skyblue')
x = np.linspace(response_times.min(), response_times.max(), 100)
axes[0, 0].plot(x, stats.norm.pdf(x, mean_rt, std_rt), 'r-', linewidth=2, label='Нормальная кривая')
axes[0, 0].axvline(mean_rt + 3*std_rt, color='red', linestyle='--', alpha=0.8, label='+3σ')
axes[0, 0].set_title('Распределение времени отклика')
axes[0, 0].set_xlabel('Время отклика, мс')
axes[0, 0].legend()

# 2. Q-Q график
stats.probplot(response_times, dist="norm", plot=axes[0, 1])
axes[0, 1].set_title('Q-Q график проверки нормальности')

# 3. Box plot с выбросами
bp = axes[1, 0].boxplot(response_times, patch_artist=True)
bp['boxes'][0].set_facecolor('lightblue')
axes[1, 0].set_title('Box plot времени отклика')
axes[1, 0].set_ylabel('Время отклика, мс')

# 4. Временной ряд с контрольными границами (если есть временные данные)
if 'timestamp' in df.columns:
    axes[1, 1].plot(df['timestamp'], df['response_time_ms'], alpha=0.6, linewidth=0.5)
    axes[1, 1].axhline(mean_rt, color='green', linestyle='-', label='Среднее')
    axes[1, 1].axhline(mean_rt + 3*std_rt, color='red', linestyle='--', label='+3σ')
    axes[1, 1].axhline(mean_rt - 3*std_rt, color='red', linestyle='--', label='-3σ')
    axes[1, 1].set_title('Контрольная карта времени отклика')
    axes[1, 1].legend()
else:
    # Если нет временных данных, показываем гистограмму логарифма
    axes[1, 1].hist(np.log(response_times), bins=30, alpha=0.7, color='orange')
    axes[1, 1].set_title('Логарифм времени отклика')
    axes[1, 1].set_xlabel('log(время отклика)')

plt.tight_layout()
plt.show()

# Рекомендации
print(f"\n💡 Рекомендации для IT-отдела:")
if skewness > 1:
    print("• Данные сильно асимметричны - много быстрых запросов, но есть очень медленные")
    print("• Рекомендуется анализ медленных запросов и оптимизация")

if len(severe_outliers) / len(response_times) > 0.01:
    print("• Слишком много аномально медленных запросов (>1%)")
    print("• Проверьте нагрузку на сервер и базу данных")

if sla_1000ms < 95:
    print(f"• SLA нарушается: только {sla_1000ms:.1f}% запросов обрабатывается быстрее 1 секунды")
    print("• Рассмотрите масштабирование инфраструктуры")

if within_3sigma < 99:
    print("• Распределение далеко от нормального")
    print("• Возможны системные проблемы или неоднородная нагрузка")
```

- ✅ Проведён полный анализ нормальности времени отклика
- ✅ Найдены аномально медленные запросы
- ✅ Рассчитаны SLA показатели
- ✅ Созданы информативные визуализации
