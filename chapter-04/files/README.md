# Учебные файлы для Главы 4

## 📂 Что здесь лежит:
- **exam-scores.csv** — результаты экзамена по аналитике (Задание 4.1)
- **bread-weights.csv** — веса булок для контроля качества (Задание 4.2)

## 📖 Как использовать файлы:

### exam-scores.csv
**Для задания 4.1**
- Содержит: имена студентов и их баллы
- Использование: анализ нормальности распределения, расчёт Z-оценок

### bread-weights.csv
**Для задания 4.2**
- Содержит: веса булок (г)
- Использование: контроль качества, поиск брака методом 3σ

### response-times.csv
**Для задания 4.3**
- Содержит: время отклика сайта в мс
- Использование: анализ нормальности, SLA расчёты

### callcenter-data.csv
**Для задания 4.4**
- Содержит: длительность звонков (сек) и оценки операторов
- Использование: сравнение операторов, поиск аномалий, связь длительности и качества

### daily-sales.csv
**Для дополнительного анализа**
- Содержит: ежедневные продажи за месяц (руб)
- Использование: анализ нормальности бизнес-данных, поиск аномалий

## 🔧 Инструкция по загрузке:

### В Excel:
1. Файл → Открыть → выберите нужный CSV файл
2. Если иероглифы:
   - Данные → Получить данные → Из текста/CSV
   - Кодировка UTF-8, разделитель ";"
3. Загрузить и закрыть редактор данных

### В Python:
```python
import pandas as pd
# Загрузка файла с указанием кодировки и разделителя
df = pd.read_csv('exam-scores.csv', encoding='utf-8', sep=';')
print(df.head())
```

### В Power BI:
1. Получить данные → Текст/CSV
2. Выберите файл, настройте разделитель и кодировку
3. Загрузите

## ❓ Частые вопросы:

- **Файл не открывается в Excel?**
  - Проверьте расширение .csv и кодировку UTF-8

- **Пустые строки при загрузке?**
  - Откройте в блокноте и удалите лишние пробелы\n
- **Python выдаёт ошибки?**
  - Попробуйте encoding='utf-8-sig' или 'cp1251'
