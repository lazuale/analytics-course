## 📚 Теоретический материал

### 4.1 Нормальное распределение — что это такое простыми словами

**Нормальное распределение — это когда большинство значений группируются вокруг среднего, а крайности встречаются редко.**

#### Жизненные примеры нормального распределения:
- **Рост людей**: большинство около 170 см, очень высокие и низкие — редко
- **Результаты ЕГЭ**: много "середнячков", мало отличников и двоечников
- **Время поездки на работу**: обычно 30-40 минут, редко 10 или 90 минут
- **Вес новорождённых**: большинство 3-4 кг, крайности редки

#### Форма нормального распределения:
```
        Частота
           ↑
           |    📊
           |  📊📊📊
           |📊📊📊📊📊
           |📊📊📊📊📊📊
           |📊📊📊📊📊📊📊
           └─────────────────→ Значения
                μ (среднее)
```

**Это знаменитая "колокольчик" или "гауссова кривая"**

### 4.2 Правило трёх сигм (68-95-99.7)

**Это самое практичное правило в статистике!**

#### Что означают проценты:
- **68%** данных лежат в пределах ±1 стандартное отклонение от среднего
- **95%** данных лежат в пределах ±2 стандартных отклонения
- **99.7%** данных лежат в пределах ±3 стандартных отклонений

#### Пример с ростом людей:
Если средний рост = 170 см, стандартное отклонение = 10 см, то:
- **68%** людей имеют рост от 160 до 180 см (170 ± 10)
- **95%** людей имеют рост от 150 до 190 см (170 ± 20)  
- **99.7%** людей имеют рост от 140 до 200 см (170 ± 30)

**Если кто-то ростом 210 см — это очень редко (вероятность < 0.15%)**

### 4.3 Практическое применение — поиск аномалий

#### Правило для бизнеса:
**Если значение выходит за ±3 сигмы — это подозрительно и требует внимания**

#### Примеры применения:

**В интернет-магазине:**
- Средние дневные продажи: 100 заказов, σ = 20
- Границы нормы: 40-160 заказов (100 ± 3×20)
- Если вдруг 200 заказов — проверить: не ошибка ли в системе?
- Если 10 заказов — что случилось с сайтом?

**В производстве:**
- Средний вес упаковки: 500г, σ = 5г
- Границы нормы: 485-515г
- Упаковка 480г — брак, нужна проверка оборудования

### 4.4 Z-оценки — универсальная "линейка" для сравнения

**Z-оценка показывает "на сколько сигм данное значение отличается от среднего"**

#### Формула (не пугайтесь!):
Z = (Значение - Среднее) / Стандартное отклонение

#### Интерпретация Z-оценок:
- **Z от -1 до +1** — нормально (68% случаев)
- **Z от -2 до +2** — довольно нормально (95% случаев)
- **Z от -3 до +3** — ещё можно считать нормальным (99.7% случаев)
- **|Z| > 3** — очень подозрительно!

#### Пример:
Ученик получил 85 баллов на экзамене, где средний балл = 70, σ = 10

Z = (85 - 70) / 10 = 1.5

**Это означает**: результат выше среднего на 1.5 стандартных отклонения — хороший результат, но не исключительный.

### 4.5 Как проверить "нормальность" данных

#### Визуальные способы:

**1. Гистограмма**
Должна выглядеть как колокольчик:
- Симметрична относительно центра
- Один чёткий пик в центре
- Хвосты затухают плавно

**2. Q-Q график (Quantile-Quantile)**
Точки должны лежать примерно на прямой линии

**3. Правило "среднее ≈ медиана"**
В нормальном распределении среднее и медиана почти равны

#### Числовые проверки:

**Коэффициент асимметрии (Skewness):**
- Близко к 0 = симметрично (хорошо)
- > 1 или < -1 = сильно асимметрично (плохо)

**Коэффициент эксцесса (Kurtosis):**
- Близко к 3 = нормально (хорошо)
- Намного больше или меньше = не нормально

### 4.6 Когда данные НЕ нормальные

#### Типичные "не нормальные" распределения:

**1. Логнормальное (зарплаты, цены недвижимости)**
- Много маленьких значений, мало больших
- Сильный "хвост" справа
- Пример: большинство получает 50-80 тыс, единицы — миллионы

**2. Равномерное (результат честной кости)**
- Все значения встречаются одинаково часто
- Прямоугольная форма распределения

**3. Биномиальное (результаты голосования)**
- Дискретные значения (только целые числа)
- Два пика в крайних точках

**4. Пуассоново (редкие события)**
- Количество звонков в call-центр
- Число аварий в день
- Асимметричное с длинным правым хвостом

#### Что делать если данные не нормальные:

**1. Трансформации:**
- **Логарифмическая**: log(x) — для данных с правым хвостом
- **Квадратный корень**: √x — для подсчётов
- **Box-Cox трансформация** — автоматический выбор

**2. Использовать непараметрические методы:**
- Медиана вместо среднего
- Перцентили вместо стандартных отклонений
- Непараметрические тесты

**3. Работать с исходными данными:**
- Понимать природу распределения
- Использовать подходящие модели

#### Функция для анализа нормальности (10 минут)

```python
def analyze_normality(data, name, ax=None):
    """Комплексный анализ нормальности распределения"""
    
    # Основные статистики
    mean_val = data.mean()
    median_val = data.median()
    std_val = data.std()
    
    # Тесты на нормальность
    skewness = stats.skew(data)
    kurtosis = stats.kurtosis(data)
    
    # Тест Шапиро-Уилка (для n < 5000)
    if len(data) <= 5000:
        shapiro_stat, shapiro_p = stats.shapiro(data)
    else:
        shapiro_stat, shapiro_p = None, None
    
    print(f"\n📊 Анализ нормальности: {name}")
    print("="*50)
    print(f"Среднее: {mean_val:.2f}")
    print(f"Медиана: {median_val:.2f}")
    print(f"Разница ср-медиана: {abs(mean_val-median_val)/mean_val*100:.1f}%")
    print(f"Стандартное отклонение: {std_val:.2f}")
    print(f"Асимметрия (skewness): {skewness:.3f}")
    print(f"Эксцесс (kurtosis): {kurtosis:.3f}")
    
    # Интерпретация
    if abs(skewness) < 0.5:
        print("✅ Асимметрия: близко к нормальному")
    elif abs(skewness) < 1:
        print("⚠️ Асимметрия: умеренно асимметричное")
    else:
        print("❌ Асимметрия: сильно асимметричное")
    
    if abs(kurtosis) < 1:
        print("✅ Эксцесс: близко к нормальному")
    else:
        print("⚠️ Эксцесс: отличается от нормального")
    
    # Правило 68-95-99.7
    within_1sigma = np.sum(np.abs(data - mean_val) <= std_val) / len(data) * 100
    within_2sigma = np.sum(np.abs(data - mean_val) <= 2*std_val) / len(data) * 100
    within_3sigma = np.sum(np.abs(data - mean_val) <= 3*std_val) / len(data) * 100
    
    print(f"\n📏 Правило трёх сигм:")
    print(f"±1σ: {within_1sigma:.1f}% (норма: 68.2%)")
    print(f"±2σ: {within_2sigma:.1f}% (норма: 95.4%)")
    print(f"±3σ: {within_3sigma:.1f}% (норма: 99.7%)")
    
    return {
        'mean': mean_val, 'median': median_val, 'std': std_val,
        'skewness': skewness, 'kurtosis': kurtosis,
        'within_1sigma': within_1sigma, 'within_2sigma': within_2sigma, 'within_3sigma': within_3sigma
    }

# Анализируем каждый тип данных
results = {}
for col in data_df.columns:
    results[col] = analyze_normality(data_df[col], col)
```

```python
# Создаём комплексную визуализацию
fig, axes = plt.subplots(3, 3, figsize=(15, 12))

columns = data_df.columns
for i, col in enumerate(columns):
    data_col = data_df[col]
    
    # Гистограмма с наложенной нормальной кривой
    axes[i, 0].hist(data_col, bins=50, density=True, alpha=0.7, color='skyblue')
    
    # Теоретическая нормальная кривая
    x = np.linspace(data_col.min(), data_col.max(), 100)
    normal_curve = stats.norm.pdf(x, data_col.mean(), data_col.std())
    axes[i, 0].plot(x, normal_curve, 'r-', linewidth=2, label='Нормальная кривая')
    axes[i, 0].set_title(f'Гистограмма: {col}')
    axes[i, 0].legend()
    
    # Q-Q график
    stats.probplot(data_col, dist="norm", plot=axes[i, 1])
    axes[i, 1].set_title(f'Q-Q график: {col}')
    
    # Box plot с отмеченными аномалиями
    axes[i, 2].boxplot(data_col)
    axes[i, 2].set_title(f'Box plot: {col}')
    
    # Отмечаем выбросы (|Z| > 3)

    if len(outliers) > 0:
```

## 📊 Работа с ненормальными данными

### 4.4 Трансформации для нормализации (20 минут)

```python
# Работаем с логнормальными данными (зарплаты)
salaries = data_df['Зарплата_тыс']

# Визуализируем исходные данные и трансформации
fig, axes = plt.subplots(2, 3, figsize=(15, 10))

transformations = [
    ('Исходные данные', salaries, None),
    ('Логарифм', np.log(salaries), 'log'),
    ('Квадратный корень', np.sqrt(salaries), 'sqrt'),
    ('Box-Cox', None, 'boxcox')  # рассчитаем отдельно
]

for i, (name, data_trans, method) in enumerate(transformations):
    row = i // 3
    col = i % 3
    
    if method == 'boxcox':
        # Box-Cox трансформация автоматически находит лучший параметр
        data_trans, lambda_param = stats.boxcox(salaries)
        name = f'Box-Cox (λ={lambda_param:.2f})'
    
    if data_trans is not None:
        # Гистограмма
        axes[row, col].hist(data_trans, bins=30, alpha=0.7, density=True)
        
        # Проверяем нормальность
        skewness = stats.skew(data_trans)
        kurtosis = stats.kurtosis(data_trans)
        
        axes[row, col].set_title(f'{name}\nSkew: {skewness:.2f}, Kurt: {kurtosis:.2f}')
        
        # Добавляем теоретическую нормальную кривую
        x = np.linspace(data_trans.min(), data_trans.max(), 100)
        normal_curve = stats.norm.pdf(x, data_trans.mean(), data_trans.std())
        axes[row, col].plot(x, normal_curve, 'r--', alpha=0.8)

# Убираем пустые подграфики
for i in range(len(transformations), 6):
    row = i // 3
    col = i % 3
    axes[row, col].axis('off')

plt.tight_layout()
plt.show()

print("\n🔄 Результаты трансформаций:")
print("Метод трансформации → Асимметрия → Оценка нормальности")
print("-" * 55)

original_skew = stats.skew(salaries)
print(f"Исходные данные    → {original_skew:6.3f} → {'❌ Не нормально' if abs(original_skew) > 0.5 else '✅ Нормально'}")

log_data = np.log(salaries)
log_skew = stats.skew(log_data)
print(f"Логарифм          → {log_skew:6.3f} → {'❌ Не нормально' if abs(log_skew) > 0.5 else '✅ Нормально'}")

sqrt_data = np.sqrt(salaries)
sqrt_skew = stats.skew(sqrt_data)
print(f"Квадратный корень → {sqrt_skew:6.3f} → {'❌ Не нормально' if abs(sqrt_skew) > 0.5 else '✅ Нормально'}")

boxcox_data, _ = stats.boxcox(salaries)
boxcox_skew = stats.skew(boxcox_data)
print(f"Box-Cox           → {boxcox_skew:6.3f} → {'❌ Не нормально' if abs(boxcox_skew) > 0.5 else '✅ Нормально'}")
```

### 4.5 Контроль качества производства (15 минут)

```python
# Симулируем данные завода по производству шоколада
# Нормальный вес плитки: 100г, допуск ±5г

np.random.seed(123)
normal_production = np.random.normal(100, 1.5, 1000)  # хорошее оборудование
defective_batch = np.random.normal(97, 3, 50)        # сбой в оборудовании

# Объединяем данные
all_weights = np.concatenate([normal_production, defective_batch])
dates = pd.date_range('2024-01-01', periods=len(all_weights), freq='H')

production_df = pd.DataFrame({
    'Время': dates,
    'Вес_грамм': all_weights
})

# Анализ контроля качества
mean_weight = production_df['Вес_грамм'].mean()
std_weight = production_df['Вес_грамм'].std()

print(f"📊 Контроль качества шоколада:")
print(f"Средний вес: {mean_weight:.2f}г")
print(f"Стандартное отклонение: {std_weight:.2f}г")

# Находим бракованные плитки (за пределами ±3σ)
z_scores = (production_df['Вес_грамм'] - mean_weight) / std_weight
outliers_mask = np.abs(z_scores) > 3
defective_count = outliers_mask.sum()

print(f"\n⚠️ Найдено бракованных плиток: {defective_count} из {len(all_weights)}")
print(f"Процент брака: {defective_count/len(all_weights)*100:.2f}%")

# Визуализация контрольной карты
plt.figure(figsize=(15, 8))

plt.subplot(2, 1, 1)
plt.plot(production_df['Время'], production_df['Вес_грамм'], alpha=0.7, linewidth=0.5)
plt.axhline(mean_weight, color='green', linestyle='-', label='Среднее')
plt.axhline(mean_weight + 3*std_weight, color='red', linestyle='--', label='+3σ')
plt.axhline(mean_weight - 3*std_weight, color='red', linestyle='--', label='-3σ')
plt.scatter(production_df['Время'][outliers_mask], 
           production_df['Вес_грамм'][outliers_mask], 
           color='red', s=50, label='Брак')
plt.title('Контрольная карта веса шоколада')
plt.ylabel('Вес, г')
plt.legend()

plt.subplot(2, 1, 2)
```
