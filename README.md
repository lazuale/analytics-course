# Analytics Course: От полного новичка до профессионала

**Математическая статистика для аналитиков без математики**

## 📚 О курсе  
Бесплатный открытый курс, шаг за шагом вводящий в аналитику данных без сложной математики. Через реальные бизнес-примеры научитесь собирать, анализировать и визуализировать данные в Excel, Power BI, Python и SQL.

### 🎯 Философия курса
- **Никаких формул** — только практика и здравый смысл  
- **От простого к сложному** — каждый шаг понятен новичку  
- **70% практики** — реальные задания на основе бизнес-кейсов  
- **Поддержка сообщества** — материалы тестируются и обновляются

---

## ⚙️ Требуемое ПО  
- Microsoft Excel 2016+ (Windows/Mac)  
- Power BI Desktop (Windows)  
- Anaconda (Python 3) + Jupyter Notebook  
- Git (для публикации на GitHub)  
- Текстовый редактор (VS Code, Notepad++)

Подробнее см. [Инструкция по установке ПО](installation-guide.md)

## 🗺 Примеры реальных кейсов
- **Анализ продаж интернет-магазина**: сегментация клиентов, ABC-анализ, выявление трендов и рекомендации по ассортименту.  
- **A/B-тестирование маркетинговой кампании**: сбор и обработка данных, проверка гипотез, расчёт статистической значимости.  
- **Прогнозирование спроса на продукт**: построение модели регрессии и оценка точности прогноза.

---

## 🛣 Дорожная карта обучения
1. **Часть I. Первые шаги (1–3 недели)**: основы аналитики, Excel, Power Query, Power Pivot, визуализация.  
2. **Часть II. Собираем и готовим данные (4–5 недели)**: импорт данных, чистка, выборки, доверительные интервалы, A/B-тесты.  
3. **Часть III. Методы анализа (6–8 недели)**: корреляция, регрессия, временные ряды, кластеризация.  
4. **Часть IV. Дашборды и отчёты (9–10 недели)**: Power BI, продвинутая визуализация, DAX.  
5. **Часть V. Python и автоматизация (11–13 недели)**: Pandas, ML, API.  
6. **Часть VI. SQL и проекты (14–16 недели)**: SQL, оптимизация, презентация результатов.

## ⚙️ Стек инструментов и главы
- **Microsoft Excel, Power Query, Power Pivot**: главы 1–3 — базовые таблицы, формулы, загрузка и трансформация данных через Power Query, продвинутый анализ через Power Pivot и сводные таблицы, гистограммы.  
- **Power BI**: главы 11–13 — построение дашбордов, интерактивность, DAX.  
- **Python**: главы 14–18 — Pandas, визуализация (Matplotlib/Seaborn), машинное обучение, API.  
- **SQL**: главы 19–22 — SELECT, JOIN, подзапросы, оптимизация, модели данных.  
- **Git и GitHub**: интеграция всех материалов, контроль версий и публикация результатов — используется во всех главах.

> Все инструменты последовательно вводятся по мере роста сложности.

---

## 🏆 Уровни сложности
- **Новичок** (главы 1–4): Excel (формулы, гистограммы), Power Query, Power Pivot, сводные таблицы.  
- **Средний** (главы 5–10): работа с данными в Excel и SQL (базовые запросы), проверка гипотез, регрессия, временные ряды, сегментация.  
- **Продвинутый** (главы 11–23): Power BI (дашборды, DAX), Python (Pandas, визуализация, ML, API), SQL (продвинутые запросы и оптимизация).

---

## 🎓 Навыки по окончанию курса
- **Анализ данных**: описательные и вероятностные методы, проверка гипотез, регрессия, временные ряды.  
- **Визуализация и трансформация**: Excel (формулы, Power Query, Power Pivot, сводные таблицы), Power BI дашборды, Matplotlib/Seaborn.  
- **Программирование**: Python (Pandas, Scikit-learn, API).  
- **Работа с данными**: SQL-запросы (SELECT, JOIN, подзапросы), оптимизация, моделирование данных.  
- **Контроль версий**: Git и GitHub.  
- **Презентация**: storytelling и оформление результатов.
