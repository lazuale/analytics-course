# Analytics Course: От полного новичка до профессионала

**Математическая статистика для аналитиков без математики**

## 📚 О курсе  
Бесплатный открытый курс, шаг за шагом вводящий в аналитику данных без сложной математики. Через реальные бизнес-примеры научитесь собирать, анализировать и визуализировать данные в Excel, Power BI, Python и SQL.

### 🎯 Философия курса
- **Никаких формул** — только практика и здравый смысл  
- **От простого к сложному** — каждый шаг понятен новичку  
- **70% практики** — реальные задания на основе бизнес-кейсов  
- **Поддержка сообщества** — материалы тестируются и обновляются
