# Analytics Course: От полного новичка до профессионала

**Математическая статистика для аналитиков без математики**

## 📚 О курсе
Курс создан для людей, которые хотят стать аналитиками данных, но никогда не работали с данными и боятся сложной математики.

### 🎯 Философия курса
- **Никаких формул** — только практика и здравый смысл
- **От простого к сложному** — каждый шаг понятен новичку  
- **70% практики** — каждая глава содержит реальные задания
- **Проверено на практике** — материалы тестируются реальным студентом

### 🛠 Стек инструментов
1. **Microsoft Excel** (версия 2016 или новее)  
2. **Power BI Desktop** (Windows)  
3. **Anaconda (Python 3)** + Jupyter Notebook  
4. **Git** и **GitHub** (для публикации упражнений)
5. **Любой текстовый редактор** (VS Code, Notepad++)

> **Все программы должны быть установлены заранее.** 
> Подробнее — [installation-guide.md](installation-guide.md)

## 🚀 Как проходить главу
1. **README.md** главы — изучите теорию и примеры
2. **practice.md** — последовательно выполните задания (4–5 упражнений)
3. **Папка files/** — скачайте необходимые CSV/XLSX/Python-файлы
4. **checklist.md** — отметьте освоение каждого пункта
5. **Перейти к следующей главе**
