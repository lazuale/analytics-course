# Analytics Course: От полного новичка до профессионала

**Математическая статистика для аналитиков без математики**

## 📚 О курсе

Этот курс создан для людей, которые хотят стать аналитиками данных, но никогда не работали с данными и боятся сложной математики. 

### 🎯 Философия курса
- **Никаких формул** — только практика и здравый смысл
- **От простого к сложному** — каждый шаг понятен новичку  
- **70% практики** — каждая глава содержит реальные задания
- **Проверено на практике** — материалы тестируются реальным студентом

### 🛠 Стек инструментов
- Excel (базовые функции и графики)
- Power Query (подготовка данных)  
- Power Pivot (работа с большими данными)
- Power BI (дашборды и визуализация)
- Python (автоматизация и продвинутая аналитика)
- SQL (работа с базами данных)

## 📋 Структура курса

### 🚀 Уровень 1: Первые шаги (Главы 1-10, 10 недель)
**Для кого**: Полные новички
**Результат**: Уверенное владение Excel, понимание основ статистики

### 📊 Уровень 2: Уверенный пользователь (Главы 11-17, 8 недель) 
**Для кого**: Освоившие основы
**Результат**: Создание дашбордов, анализ связей, сегментация

### 🔬 Уровень 3: Профессионал (Главы 18-23, 6 недель)
**Для кого**: Готовые стать аналитиками данных  
**Результат**: Комплексные решения, автоматизация, прогнозирование

## 🎓 Как проходить курс

### Рекомендуемый график:
- **1 глава = 1 неделя** для комфортного усвоения
- **2-3 часа занятий в неделю** (теория + практика)  
- **Выполняйте ВСЕ практические задания** — без них курс не работает

### Что нужно для успеха:
- Компьютер с Windows (для Power BI)
- Желание учиться и экспериментировать
- Терпение — некоторые вещи станут понятны не сразу
