# Учебные файлы для практики

## 📂 Что здесь лежит:
- **cafe-sales.csv** - данные кафе для Excel и Power BI  
- **python-first.py** - первая программа на Python
- **README.md** - этот файл с описанием

## 📖 Как использовать файлы:

### cafe-sales.csv
**Для заданий 1.3 и 1.4**
1. Скачайте файл (кнопка "Download" или "Raw")
2. Откройте в Excel для создания таблицы и графика
3. Загрузите в Power BI для создания дашборда

**Если Excel показывает кракозябры:**
- Откройте файл в блокноте
- Сохранить как → кодировка UTF-8
- Теперь откройте в Excel

### python-first.py  
**Для задания 1.5**
1. Скачайте файл 
2. Откройте в Jupyter Notebook или любом текстовом редакторе
3. Копируйте код по частям и выполняйте
