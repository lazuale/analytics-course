## 📚 Теоретический материал

### 1.1 Зачем нужна аналитика данных
Аналитика данных — это способ принимать решения на основе фактов, а не интуиции.

**Пример:**
- Без данных: упаковка красная, продажи упали на 30%  
- С данными: опрос 500 клиентов, 68% предпочитают синий, продажи выросли на 25%

### 1.3 Основные формулы Excel
Чтобы выполнять задания в Excel, используйте следующие функции:
- Сумма: `=СУММ(диапазон)` — суммирует числа  
- Среднее: `=СРЗНАЧ(диапазон)` — вычисляет среднее арифметическое  
- Минимум/Максимум: `=МИН(диапазон)`, `=МАКС(диапазон)`  
- Деление: в ячейке `=B2/C2` — для расчёта средней цены  

**Построение гистограммы:**
1. Выделите столбец с данными.  
2. В меню Excel выберите **Вставка → Диаграммы → Гистограмма**.  

### 1.4 Базовые операции в Python
В Jupyter Notebook или скрипте используйте:
```python
import csv
from statistics import mean
# Загрузка данных из CSV
with open('files/cafe-sales.csv', encoding='utf-8') as f:
    reader = csv.DictReader(f, delimiter=';')
    cups = [int(row['Чашек']) for row in reader]
# Расчёт среднего
avg = mean(cups)
print('Среднее число чашек:', avg)
```
