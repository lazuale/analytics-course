## 💻 Задание 1.3: Первая таблица в Excel
**Время выполнения: 30 минут**

### Что создаём:
Таблицу продаж кофе в вашем воображаемом кафе за неделю

### Пошаговая инструкция:

#### Шаг 1: Создаём таблицу (10 минут)
1. **Откройте Excel**
2. **Создайте заголовки** в первой строке:
   - A1: Дата
   - B1: День недели  
   - C1: Чашек кофе
   - D1: Выручка (руб)
   - E1: Средняя цена

3. **Заполните данные:**

| A | B | C | D | E |
|---|---|---|---|---|
| Дата | День недели | Чашек кофе | Выручка | Средняя цена |
| 22.01.2024 | Понедельник | 45 | 13500 | =D2/C2 |
| 23.01.2024 | Вторник | 52 | 15600 | =D3/C3 |
| 24.01.2024 | Среда | 38 | 11400 | =D4/C4 |
| 25.01.2024 | Четверг | 41 | 12300 | =D5/C5 |
| 26.01.2024 | Пятница | 67 | 20100 | =D6/C6 |
| 27.01.2024 | Суббота | 89 | 26700 | =D7/C7 |
| 28.01.2024 | Воскресенье | 73 | 21900 | =D8/C8 |

#### Шаг 2: Добавляем итоги (5 минут)
1. **В строке 9 создайте итоги:**
   - A9: ИТОГО
   - C9: `=СУММ(C2:C8)`
   - D9: `=СУММ(D2:D8)`
   - E9: `=D9/C9`

2. **В строке 10 средние значения:**
   - A10: СРЕДНЕЕ
   - C10: `=СРЗНАЧ(C2:C8)`
   - D10: `=СРЗНАЧ(D2:D8)`

#### Шаг 3: Форматируем таблицу (5 минут)
1. **Выделите всю таблицу** (A1:E10)
2. **Вставка → Таблица** (или Ctrl+T)
3. **Поставьте галочку** "Таблица с заголовками"
4. **Выберите стиль** таблицы

#### Шаг 4: Создаём график (10 минут)
1. **Выделите столбцы B и C** (День недели и Чашек кофе)
2. **Вставка → Диаграммы → Гистограмма**
3. **Добавьте заголовок** графика: "Продажи кофе по дням недели"
4. **Измените цвета** столбцов на ваш вкус

## 🐍 Задание 1.5: Первая программа на Python
**Время выполнения: 15 минут**

### Что создаём:
Простую программу для анализа данных кафе

### Пошаговая инструкция:

#### Шаг 1: Открываем Jupyter Notebook (3 минуты)
1. **Пуск → Anaconda Navigator**
2. **Нажмите Launch** напротив "Jupyter Notebook"
3. **Создайте новый файл:** New → Python 3

#### Шаг 2: Пишем первую программу (12 минут)

**Ячейка 1: Импорт данных**
```python
# Данные нашего кафе за неделю
coffee_cups = [45, 52, 38, 41, 67, 89, 73]
revenue = [13500, 15600, 11400, 12300, 20100, 26700, 21900]
days = ["Понедельник", "Вторник", "Среда", "Четверг", "Пятница", "Суббота", "Воскресенье"]

print("🏪 Добро пожаловать в аналитику кафе!")
print("📊 Данные загружены успешно!")
```

**Ячейка 2: Базовая статистика**
```python
# Считаем основные показатели
total_cups = sum(coffee_cups)
total_revenue = sum(revenue)
average_cups = total_cups / len(coffee_cups)
average_revenue = total_revenue / len(revenue)
average_price = total_revenue / total_cups

print(f"☕ Всего продано чашек: {total_cups}")
print(f"💰 Общая выручка: {total_revenue:,} руб")
print(f"📈 Средние чашки в день: {average_cups:.1f}")
print(f"💵 Средняя выручка в день: {average_revenue:,.0f} руб")
print(f"🏷️ Средняя цена чашки: {average_price:.0f} руб")
```

**Ячейка 3: Поиск лучшего и худшего дня**
```python
# Находим рекордсменов
best_day_index = coffee_cups.index(max(coffee_cups))
worst_day_index = coffee_cups.index(min(coffee_cups))

print(f"🏆 Лучший день: {days[best_day_index]} ({max(coffee_cups)} чашек)")
print(f"😔 Худший день: {days[worst_day_index]} ({min(coffee_cups)} чашек)")

# Считаем разницу
difference = max(coffee_cups) - min(coffee_cups)
print(f"📊 Разница между лучшим и худшим днём: {difference} чашек")
```

**Ячейка 4: Простая визуализация**
```python
# Создаём простой график из символов
print("\n📊 График продаж (каждый * = 10 чашек):")
for i, day in enumerate(days):
    stars = "*" * (coffee_cups[i] // 10)
    print(f"{day[:3]}: {stars} ({coffee_cups[i]})")
```

**Ячейка 5: Выводы**
```python
# Делаем простые выводы
print("\n🤔 Простые выводы:")

if max(coffee_cups) > average_cups * 1.5:
    best_day = days[best_day_index]
    print(f"• {best_day} - явно выдающийся день! Стоит изучить причины.")

weekend_sales = coffee_cups[5] + coffee_cups[6]  # Суббота + Воскресенье
weekday_sales = sum(coffee_cups[:5])  # Пн-Пт

if weekend_sales > weekday_sales / 5 * 2:
    print("• Выходные дни более прибыльны чем будни.")
else:
    print("• Будние дни стабильнее выходных.")

if average_price > 300:
    print("• Средняя цена выше 300 рублей - премиум сегмент.")
else:
    print("• Демократичные цены - массовый сегмент.")
```

### Как запускать:
1. **В каждой ячейке нажимайте Shift + Enter**
2. **Программа выполнится и покажет результат**
3. **Ячейки выполняются по порядку**

- ✅ Программа выводит базовую статистику
- ✅ Находит лучший и худший день
- ✅ Рисует простой график символами
- ✅ Делает автоматические выводы
