# Глоссарий ключевых терминов

Этот файл содержит определения и объяснения основных терминов и сокращений, используемых в курсе.

## А
аналитика данных — процесс сбора, обработки, анализа и визуализации данных для принятия обоснованных решений.
ассортимент — набор продуктов или услуг, предлагаемых бизнесом.

## Б
бизнес-аналитика — аналитика, направленная на улучшение бизнес-процессов и повышение эффективности.
бутстрэппинг — метод оценки распределения статистики через повторную выборку.

## В
выбросы — экстремальные значения, сильно отличающиеся от основной массы данных.
визуализация — графическое представление данных (графики, диаграммы).

## Д
dax (Data Analysis Expressions) — формульный язык для вычислений в Power BI и Excel Power Pivot.
диаграмма рассеяния — график, показывающий связь между двумя числовыми переменными.

## И
интерактивность — возможность управлять визуализациями (фильтры, срезы).

## К
когортный анализ — метод анализа групп объектов, объединённых по дате начала наблюдения.
квадратный корень — математическая функция sqrt(x), используется для трансформации данных.

## М
медиана — серединное значение упорядоченного набора данных.
модель данных — логическая структура связанных таблиц в Power BI и Excel Power Pivot.
мода — наиболее часто встречающееся значение.

## Н
нормальное распределение — симметричное распределение в форме колокольчика (гауссово).

## П
перцентили — значения, делящие данные на 100 равных частей (например, 25-й процентиль).
процентиль — отдельное значение в распределении данных, соответствующее заданному проценту.

## Р
регрессия — метод анализа зависимости одной переменной от другой.
среднее арифметическое — сумма значений, деленная на их количество.

## С
стандартное отклонение — мера разброса данных вокруг среднего.
скрипт — файл с кодом (например, Python). 
сводная таблица — инструмент Excel для группировки и агрегирования данных.

## Т
трёх сигм (3σ) правило — метод поиска аномалий: значений вне ±3σ считаются выбросами.
тест Шапиро-Уилка — статистический тест нормальности распределения.

## U
UTF-8 — рекомендуемая кодировка для всех текстовых и CSV файлов.
