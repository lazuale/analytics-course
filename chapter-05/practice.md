### 📝 Пошаговые инструкции

#### Этап 1: Установка PostgreSQL

**Для Windows:**
1. Скачайте PostgreSQL с официального сайта: https://www.postgresql.org/download/windows/
2. Запустите installer и выберите версию PostgreSQL 15 или выше
3. При установке запомните пароль для пользователя `postgres`
4. Установите порт по умолчанию: `5432`
5. Убедитесь, что установился pgAdmin (графический интерфейс)

**Для macOS:**
1. Установите через Homebrew: `brew install postgresql`
2. Запустите сервис: `brew services start postgresql`
3. Создайте пользователя: `createuser -s postgres`

**Для Linux (Ubuntu/Debian):**
```bash
sudo apt update
sudo apt install postgresql postgresql-contrib
sudo systemctl start postgresql
sudo systemctl enable postgresql
```

#### Этап 2: Первое подключение и создание базы данных

1. **Откройте pgAdmin** или подключитесь через командную строку:
   ```bash

   ```

2. **Создайте новую базу данных для учебного проекта:**
   ```sql
   CREATE DATABASE analytics_course;
   ```

3. **Подключитесь к созданной базе данных:**
   ```sql
   \c analytics_course;
   ```

4. **Проверьте подключение:**
   ```sql
   SELECT current_database(), current_user, version();
   ```

#### Этап 3: Создание первой таблицы

1. **Создайте простую таблицу для тестирования:**
   ```sql

   ```

3. **Проверьте данные:**

#### Этап 4: Изучение основных команд PostgreSQL

1. **Просмотрите список баз данных:**

#### Этап 2: Создание таблиц с ограничениями

1. **Создайте таблицу категорий:**
   ```sql
   CREATE TABLE categories (
       category_id SERIAL PRIMARY KEY,
       category_name VARCHAR(100) NOT NULL UNIQUE,
       description TEXT,
       created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
   );
   ```

2. **Создайте таблицу клиентов:**
   ```sql
   CREATE TABLE customers (
       customer_id SERIAL PRIMARY KEY,
       first_name VARCHAR(50) NOT NULL,
       last_name VARCHAR(50) NOT NULL,
       email VARCHAR(100) NOT NULL UNIQUE,
       phone VARCHAR(20),
       address TEXT,
       city VARCHAR(50),
       registration_date DATE DEFAULT CURRENT_DATE,
       is_active BOOLEAN DEFAULT TRUE
   );
   ```

3. **Создайте таблицу товаров:**
   ```sql
   CREATE TABLE products (
       product_id SERIAL PRIMARY KEY,
       product_name VARCHAR(200) NOT NULL,
       description TEXT,
       category_id INTEGER NOT NULL,
       price DECIMAL(10,2) NOT NULL CHECK (price > 0),
       stock_quantity INTEGER DEFAULT 0 CHECK (stock_quantity >= 0),
       is_active BOOLEAN DEFAULT TRUE,
       created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
       FOREIGN KEY (category_id) REFERENCES categories(category_id)
   );
   ```

4. **Создайте таблицу заказов:**
   ```sql
   CREATE TABLE orders (
       order_id SERIAL PRIMARY KEY,
       customer_id INTEGER NOT NULL,
       order_date DATE DEFAULT CURRENT_DATE,
       status VARCHAR(20) DEFAULT 'pending' CHECK (
           status IN ('pending', 'processing', 'shipped', 'delivered', 'cancelled')
       ),
       total_amount DECIMAL(12,2) DEFAULT 0 CHECK (total_amount >= 0),
       shipping_address TEXT,
       created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
       FOREIGN KEY (customer_id) REFERENCES customers(customer_id)
   );
   ```

5. **Создайте таблицу позиций заказов:**
   ```sql
   CREATE TABLE order_items (
       item_id SERIAL PRIMARY KEY,
       order_id INTEGER NOT NULL,
       product_id INTEGER NOT NULL,
       quantity INTEGER NOT NULL CHECK (quantity > 0),
       unit_price DECIMAL(10,2) NOT NULL CHECK (unit_price > 0),
       total_price DECIMAL(12,2) GENERATED ALWAYS AS (quantity * unit_price) STORED,
       FOREIGN KEY (order_id) REFERENCES orders(order_id),
       FOREIGN KEY (product_id) REFERENCES products(product_id),
       UNIQUE(order_id, product_id)
   );
   ```
