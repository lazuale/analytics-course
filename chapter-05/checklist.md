### 🛠 SQL синтаксис и команды

Отметьте ✅, если вы **знаете и понимаете:**

- [ ] **CREATE TABLE** — синтаксис создания таблиц с различными типами данных
- [ ] **Ограничения целостности** — NOT NULL, UNIQUE, CHECK, PRIMARY KEY, FOREIGN KEY
- [ ] **SELECT** — базовый синтаксис выборки данных из таблиц
- [ ] **WHERE** — различные типы условий фильтрации (=, >, <, LIKE, IN, BETWEEN, IS NULL)
- [ ] **ORDER BY** — сортировка результатов по одному и нескольким полям
- [ ] **LIMIT и OFFSET** — ограничение количества результатов и пагинация
- [ ] **Агрегатные функции** — COUNT, SUM, AVG, MIN, MAX для анализа данных
- [ ] **GROUP BY и HAVING** — группировка данных и фильтрация групп

- [ ] **Нормализация данных** — принципы разделения данных на связанные таблицы
- [ ] **Типы данных** — когда использовать INTEGER, VARCHAR, TEXT, DATE, DECIMAL

## 🛠 Практические навыки

### 🔧 Работа с PostgreSQL

Отметьте ✅, если вы **умеете выполнять без подсказок:**

#### Установка и администрирование:
- [ ] **Установить PostgreSQL** — на свою операционную систему
- [ ] **Подключиться к серверу** — через psql или pgAdmin
- [ ] **Создать базу данных** — и подключиться к ней
- [ ] **Использовать базовые команды psql** — \l, \dt, \d, \c, \help
- [ ] **Управлять пользователями и правами** — создание пользователей и назначение прав

#### Создание структуры данных:
- [ ] **Создать таблицы** — с правильными типами данных и ограничениями
- [ ] **Определить первичные ключи** — SERIAL PRIMARY KEY
- [ ] **Создать внешние ключи** — REFERENCES для связывания таблиц
- [ ] **Добавить ограничения CHECK** — для контроля корректности данных
- [ ] **Создать индексы** — для оптимизации часто используемых запросов
- [ ] **Создать представления (VIEW)** — для упрощения доступа к данным

#### Работа с данными:
- [ ] **Добавлять данные** — INSERT с проверкой ограничений
- [ ] **Обновлять записи** — UPDATE с условиями WHERE
- [ ] **Удалять данные** — DELETE с безопасными условиями
- [ ] **Импортировать данные** — из CSV файлов

### 📊 Написание SQL-запросов

Отметьте ✅, если вы **можете самостоятельно написать:**

#### Базовые запросы:
- [ ] **Простой SELECT** — выбор всех или определенных столбцов
- [ ] **SELECT с WHERE** — фильтрация по точным значениям
- [ ] **Поиск по шаблону** — LIKE с wildcards (%, _)
- [ ] **Диапазон значений** — BETWEEN для числовых и дат
- [ ] **Проверка на множество** — IN для списка значений
- [ ] **Работа с NULL** — IS NULL, IS NOT NULL, COALESCE
- [ ] **Сортировка** — ORDER BY по одному и нескольким полям
- [ ] **Ограничение результатов** — LIMIT и OFFSET

#### Аналитические запросы:
- [ ] **Подсчет записей** — COUNT(*), COUNT(column)
- [ ] **Суммирование** — SUM для числовых полей
- [ ] **Средние значения** — AVG с округлением
- [ ] **Минимум и максимум** — MIN, MAX для различных типов данных
- [ ] **Группировка** — GROUP BY по одному и нескольким полям
- [ ] **Фильтрация групп** — HAVING с агрегатными условиями
- [ ] **Сложные условия** — комбинирование AND, OR, NOT
