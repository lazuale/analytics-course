## 📚 Теоретический материал

### 2.1 Три главных вопроса о любых данных

Когда аналитик смотрит на данные, он всегда задаёт себе **три простых вопроса**:

1. **Что типично?** (центральные значения)
2. **Насколько разбросаны данные?** (разброс значений)  
3. **Есть ли что-то необычное?** (выбросы и аномалии)

Ответив на эти вопросы, мы получаем **полный портрет** наших данных.

### 2.2 Что типично? — Меры центральной тенденции

#### Среднее арифметическое — "как обычно"

**Простыми словами:** Сложили всё и поделили на количество

**Когда использовать:**
- Зарплаты в компании (если нет супер-высоких зарплат)
- Средние продажи
- Время выполнения задач

**Когда НЕ использовать:**
- Если есть очень большие или очень маленькие значения
- Доходы населения (богатые сильно "перетягивают")

**Жизненный пример:**
В кафе продали кофе: 20, 25, 30, 22, 28 чашек
Среднее = (20+25+30+22+28) ÷ 5 = 25 чашек в день

#### Медиана — "значение посередине"

**Простыми словами:** Выстроили по порядку, взяли то что в центре

**Когда использовать:**
- Доходы населения
- Цены на недвижимость  
- Любые данные с "выбросами"

**Как считать:**
1. Расставляем по порядку: 20, 22, 25, 28, 30
2. Берём средний: медиана = 25

**Жизненный пример:**
Зарплаты в отделе: 50к, 55к, 60к, 65к, 250к
Среднее = 96к (обманчиво высоко из-за директора)
Медиана = 60к (честно показывает типичную зарплату)

#### Мода — "самое частое"

**Простыми словами:** Что встречается чаще всего

**Когда использовать:**
- Самый популярный размер одежды
- Частые проблемы в тех. поддержке
- Любые категории и группы

**Жизненный пример:**
Размеры обуви в магазине: 36, 37, 38, 38, 38, 39, 40, 41
Мода = 38 (встречается 3 раза)

### 2.3 Насколько разбросаны? — Меры разброса

#### Размах — "от самого маленького до самого большого"

**Простыми словами:** Максимум минус минимум

**Пример:** Температура за неделю: +2°C, +5°C, +8°C, +12°C
Размах = 12° - 2° = 10°C

#### Стандартное отклонение — "насколько данные отличаются от среднего"

**Простыми словами:** Показывает, как далеко обычно находятся значения от среднего

**Интуитивное понимание:**
- Маленькое отклонение = данные кучно, предсказуемо
- Большое отклонение = данные разбросаны, хаотично

**Практическое правило:**
- ~68% данных лежат в пределах ±1 стандартное отклонение от среднего
- ~95% данных лежат в пределах ±2 стандартных отклонения

### 2.4 Что необычного? — Выбросы и аномалии

#### Как найти выбросы простым способом

**Правило "1,5 × межквартильный размах":**
1. Найдите 25-й процентиль (Q1) и 75-й процентиль (Q3)
2. Считайте межквартильный размах: IQR = Q3 - Q1  
3. Выбросы: всё что меньше Q1 - 1,5×IQR или больше Q3 + 1,5×IQR

**Зачем искать выбросы:**
- Ошибки в данных
- Необычные события, требующие внимания
- Особые случаи, которые нужно изучить отдельно

### Задание 2.3: Тот же анализ в Python (20 минут)

#### Шаг 1: Загрузка данных (5 минут)

```python
import pandas as pd
import numpy as np

# Данные зарплат
salaries = [85, 90, 75, 120, 95, 88, 200, 82, 78, 300]
names = ['Иванов', 'Петров', 'Сидоров', 'Козлов', 'Морозов', 
         'Волков', 'Зайцев', 'Лисин', 'Орлов', 'Беркутов']

# Создаём DataFrame (таблицу)
df = pd.DataFrame({
    'Сотрудник': names,
    'Зарплата': salaries
})

print("📊 Данные загружены:")
print(df)
```

#### Шаг 2: Основная статистика (10 минут)

```python
# Быстрая сводка всех показателей
print("\n📈 Описательная статистика:")
print(df['Зарплата'].describe())

# Отдельные показатели с объяснениями
salary_mean = df['Зарплата'].mean()
salary_median = df['Зарплата'].median()
salary_std = df['Зарплата'].std()
salary_min = df['Зарплата'].min()
salary_max = df['Зарплата'].max()

print(f"\n💰 Основные показатели:")
print(f"Среднее: {salary_mean:.0f} тыс руб")
print(f"Медиана: {salary_median:.0f} тыс руб")
print(f"Стандартное отклонение: {salary_std:.0f} тыс руб")
print(f"Размах: {salary_min} - {salary_max} тыс руб")
```

#### Шаг 3: Поиск выбросов (5 минут)

```python
# Находим выбросы методом межквартильного размаха
Q1 = df['Зарплата'].quantile(0.25)
Q3 = df['Зарплата'].quantile(0.75)
IQR = Q3 - Q1

# Границы для выбросов
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR

print(f"\n🔍 Поиск выбросов:")
print(f"Нижняя граница: {lower_bound:.0f}")
print(f"Верхняя граница: {upper_bound:.0f}")

# Находим выбросы
outliers = df[(df['Зарплата'] < lower_bound) | (df['Зарплата'] > upper_bound)]
print(f"\n⚠️ Найденные выбросы:")
print(outliers)
```

## 🏪 Практический кейс: Анализ продаж интернет-магазина

### Задание 2.5: Комплексный анализ (30 минут)

Представим данные интернет-магазина за месяц:

```python
# Данные магазина
import pandas as pd
import numpy as np

# Создаём реалистичные данные
np.random.seed(42)  # для воспроизводимости

days = list(range(1, 31))
orders = np.random.poisson(25, 30)  # среднее 25 заказов в день
revenue = orders * np.random.normal(2000, 500, 30)  # средний чек ~2000 руб

# Добавляем несколько "особых" дней
orders[14] = 45  # выходной день - больше заказов
orders[28] = 8   # технические проблемы - мало заказов
revenue[14] = orders[14] * 2200  # в выходной средний чек выше
revenue[28] = orders[28] * 1800  # в проблемный день чек ниже

shop_data = pd.DataFrame({
    'День': days,
    'Заказов': orders,
    'Выручка': revenue.round(0),
    'Средний_чек': (revenue / orders).round(0)
})

print("🛒 Данные интернет-магазина за месяц:")
print(shop_data.head(10))
```

#### Анализ по каждому показателю:

```python
# Функция для красивого анализа любого показателя
def analyze_metric(data, metric_name, unit=""):
    print(f"\n📊 Анализ показателя: {metric_name}")
    print("="*50)
    
    mean_val = data.mean()
    median_val = data.median()
    std_val = data.std()
    min_val = data.min()
    max_val = data.max()
    
    print(f"Среднее: {mean_val:.0f} {unit}")
    print(f"Медиана: {median_val:.0f} {unit}")
    print(f"Стандартное отклонение: {std_val:.0f} {unit}")
    print(f"Диапазон: {min_val:.0f} - {max_val:.0f} {unit}")
    
    # Простая интерпретация
    if std_val / mean_val > 0.3:
        print("💡 Данные довольно разбросаны - есть изменчивость")
    else:
        print("💡 Данные стабильные - мало вариативности")
    
    if abs(mean_val - median_val) / mean_val > 0.1:
        print("⚠️ Среднее сильно отличается от медианы - есть выбросы")
    else:
        print("✅ Среднее близко к медиане - данные симметричны")

# Анализируем каждый показатель
analyze_metric(shop_data['Заказов'], "Количество заказов", "шт")
analyze_metric(shop_data['Выручка'], "Выручка", "руб")
analyze_metric(shop_data['Средний_чек'], "Средний чек", "руб")
```
