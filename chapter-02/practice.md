**Вариант A - Excel:**

```python
import pandas as pd
import numpy as np

# Создаём реалистичные данные
np.random.seed(42)
days = list(range(1, 31))

# Генерируем данные с некоторой логикой
base_orders = 25
orders = np.random.poisson(base_orders, 30)

# Выходные дни - больше заказов
weekend_days = [5, 6, 12, 13, 19, 20, 26, 27]  # примерные выходные
for day in weekend_days:
    if day <= 30:
        orders[day-1] = int(orders[day-1] * 1.3)

revenue = (orders * avg_check / 1000).round(1)  # в тысячах
avg_check = (revenue * 1000 / orders).round(0)

# Создаём DataFrame
shop_data = pd.DataFrame({
    'День': days,
    'Заказов': orders,
    'Выручка_тыс': revenue,
    'Средний_чек': avg_check
})

print(shop_data.head(10))
```

### Шаг 2: Описательная статистика (15 минут)

**Для каждого показателя рассчитайте:**
- Среднее значение
- Медиану  
- Стандартное отклонение
- Минимум и максимум
- Коэффициент вариации (стд.откл / среднее × 100%)

**В Python:**
```python
# Функция для анализа любого показателя
def analyze_metric(data, metric_name, unit=""):
    stats = {
        'Среднее': data.mean(),
        'Медиана': data.median(),
        'Стд.откл': data.std(),
        'Минимум': data.min(),
        'Максимум': data.max(),
        'Коэф.вар': (data.std() / data.mean() * 100)
    }
    
    print(f"\n📊 {metric_name}:")
    for key, value in stats.items():
        if key == 'Коэф.вар':
            print(f"{key}: {value:.1f}%")
        else:
            print(f"{key}: {value:.0f} {unit}")
    
    return stats

# Анализируем каждый показатель
orders_stats = analyze_metric(shop_data['Заказов'], "Количество заказов в день", "шт")
revenue_stats = analyze_metric(shop_data['Выручка_тыс'], "Ежедневная выручка", "тыс руб")
check_stats = analyze_metric(shop_data['Средний_чек'], "Средний чек", "руб")
```

### Частые ошибки:

1. **Использование среднего при наличии выбросов**
   - Решение: всегда сравнивайте среднее с медианой

2. **Игнорирование выбросов**

3. **Анализ без контекста**
   - Решение: всегда связывайте цифры с бизнес-ситуацией

   - СРЗНАЧ() - среднее

   - СТАНДОТКЛОН.В() - стандартное отклонение для выборки

### Дополнительные функции Excel:

- `ПРОЦЕНТИЛЬ(массив; 0,25)` - первый квартиль
- `ПРОЦЕНТИЛЬ(массив; 0,75)` - третий квартиль
- `СЧЁТ(диапазон)` - количество чисел
- `МАКС(диапазон)` и `МИН(диапазон)` - максимум и минимум
